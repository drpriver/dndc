To simplify my life, this program requires Python to be installed
on the system and for certain packages to be installed as well.

Directions:
-----------

Go to https://www.python.org/downloads/windows/ and download the windows
installer, or just go to this url to download the installer:
https://www.python.org/ftp/python/3.9.2/python-3.9.2-amd64.exe

Use the installer to install Python.

You should be able to run this program by double-clicking pygdndc.pyw.
The first time, it will ask you to install its dependencies. Click 'Yes'.
