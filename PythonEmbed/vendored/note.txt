These Python files were copied from the Python3.8 standard library,
with light modifications to enable them to be frozen into
a minimal, embedded executable. Where possible, I have noted
in the .py file with the comment #FROZENHACKS as to what
I have changed or commented out. Primarily I disabled code that wouldn't
make sense in an embedded app or which I did not want to bundle.

I claim no authorship of these files.
